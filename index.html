狀態提醒

銑刀預設表

單位：Vc = m/min，fz = mm/tooth。這是「起刀預設」，不是廠商極限。S 會自動受最高轉速限制。

鑽頭預設表

單位：Vc = m/min，fn = mm/rev。fn 用「鑽徑比例」估算，例如 fn = D×1.8%。

啄鑽 Q 建議表

這是程式內建的啄鑽邏輯：先看孔深倍率，再依材料、鑽頭材質與冷卻方式修正。Q 是 G83/G73 每次鑽深，實際仍要看切屑是否順利排出。

孔深倍率預設策略Q 起始值備註
≤ 2DG81 直鑽不啄淺孔通常不需要 G83；排屑差才改 G73。
2D ~ 3DG81 / G73 輕斷屑約 0.8D ~ 1.2D視材料與切屑形狀決定。
3D ~ 5DG83 啄鑽約 0.7D ~ 1.0D一般深孔起點，Q 不宜盲目過大。
5D ~ 8DG83 深孔約 0.45D ~ 0.75D加強退屑與冷卻，必要時降低 fn。
> 8D長鑽 / 深孔策略約 0.3D ~ 0.5D建議專用長鑽、導孔、內冷或分段加工。
